# Hi there, I'm Ibrahim Sudheysi Ali 👋

Fourth-year Informatics & Computer Science student at Strathmore University with a parallel cybersecurity track at Moringa School. I build production systems end-to-end — backend schemas, mobile clients, web dashboards — and I'm drawn to the hard architectural problems: multi-tenancy, offline-first sync, on-device inference, and privacy-by-design. I like reducing messy operational reality to clean, well-typed systems that actually ship.

- 🔭 **Current Focus:** My final-year project, **CAMR** (Capability-Adaptive Memory Retrieval) — a frozen-weights, token-compressed associative memory engine that lets small on-device LLMs punch above their weight by retrieving compressed context instead of relying on raw window size.
- 🌱 **Learning & Exploration:** On-device AI (Apple Foundation Models, Gemma 3n, Llama 3.2), retrieval-and-summarization under tight context budgets, RLS-based tenant isolation, and offensive security (RCE exploitation, post-exploitation tradecraft).
- ⚡ **Core Interests:** Multi-tenant SaaS · edge/on-device AI · Android & iOS · penetration testing · privacy-first architecture.
